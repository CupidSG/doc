Contrat
~~~~~~~

Dans cet onglet, il est possible d'établir un lien entre un objet **Contrat** présent dans GLPI et le matériel.

.. note::
	Le lien est établi dans les deux sens. Il sera visible depuis le matériel et depuis le contrat.
